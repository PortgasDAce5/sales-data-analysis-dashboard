# 📊 Sales Data Analysis Dashboard

This project is a data analysis dashboard built using Python and Streamlit to analyze supermarket sales data. It provides insights into sales performance across different branches and product categories.

---

## 🚀 Features

- Interactive dashboard using Streamlit
- Filter data by branch and product line
- View total sales metrics
- Visualize sales by payment method
- Analyze sales trends over time

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Streamlit

---

## 📁 Dataset

The dataset used is a supermarket sales dataset containing information about:
- Sales
- Product categories
- Payment methods
- Branch performance
