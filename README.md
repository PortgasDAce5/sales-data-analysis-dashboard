# 📊 Sales Data Analysis Dashboard

This project is a data analysis dashboard built using Python and Streamlit to analyze supermarket sales data. It provides insights into sales performance across different branches and product categories.

---

## 🚀 Features

* Interactive dashboard using Streamlit
* Filter data by branch and product line
* View total sales metrics
* Visualize sales by payment method
* Analyze sales trends over time

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Streamlit

---

## 📁 Dataset

The dataset used is a supermarket sales dataset containing information about:

* Sales
* Product categories
* Payment methods
* Branch performance

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/PortgasDAce5/sales-data-dashboard.git

2. Navigate to the folder:
   cd sales-data-dashboard

3. Install required libraries:
   pip install pandas matplotlib streamlit

4. Run the app:
   streamlit run app.py

---

## 📌 Project Objective

The goal of this project is to perform exploratory data analysis and present insights through an interactive dashboard to support data-driven decision-making.
